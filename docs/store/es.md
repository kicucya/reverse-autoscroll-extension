Reverse AutoScroll lleva el desplazamiento natural de macOS al desplazamiento automático con clic central de Chrome.

Al hacer clic central para activar el desplazamiento automático, mover el ratón hacia abajo desplaza la página HACIA ARRIBA, siguiendo la dirección de "el contenido sigue al dedo" que los usuarios de macOS esperan.

Características:
- Control independiente por eje: invierte el vertical, el horizontal o ambos
- Curva de aceleración con zona muerta de sensación nativa
- Indicador visual limpio en la posición del clic
- Actívalo y desactívalo con clic central (no es necesario mantener pulsado)
- Sal en cualquier momento con Esc, cualquier clic o al cambiar de ventana
- Cero recopilación de datos: la configuración se almacena solo de forma local

Ideal para:
- Usuarios de macOS que activaron el desplazamiento natural en todo el sistema
- Cualquier persona que alterne entre Mac y Windows
- Usuarios que encuentran la dirección del desplazamiento automático tradicional poco intuitiva

Funciona en todas las páginas web normales (http/https). No se ejecuta en páginas chrome:// ni en Chrome Web Store.
