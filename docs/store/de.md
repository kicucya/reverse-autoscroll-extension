Reverse AutoScroll bringt das natürliche Scrollen von macOS in die Autoscroll-Funktion von Chrome per mittlerer Maustaste.

Wenn Sie per Mittelklick das Autoscrollen aktivieren, scrollt die Seite beim Bewegen der Maus nach unten NACH OBEN — passend zur „Inhalt folgt dem Finger"-Richtung, die macOS-Nutzer gewohnt sind.

Funktionen:
- Unabhängige Achsensteuerung — vertikal, horizontal oder beides umkehren
- Natürlich wirkende Beschleunigungskurve mit Totzone
- Dezente visuelle Anzeige an der Klickposition
- Ein-/Ausschalten per Mittelklick (kein Gedrückthalten nötig)
- Jederzeit beenden mit Esc, beliebigem Klick oder Fensterwechsel
- Keine Datenerfassung — Einstellungen werden nur lokal gespeichert

Ideal für:
- macOS-Nutzer, die natürliches Scrollen systemweit aktiviert haben
- Alle, die zwischen Mac und Windows wechseln
- Nutzer, denen die herkömmliche Autoscroll-Richtung unlogisch vorkommt

Funktioniert auf allen regulären Webseiten (http/https). Läuft nicht auf chrome://-Seiten oder im Chrome Web Store.
