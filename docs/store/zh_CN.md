Reverse AutoScroll 为 Chrome 的鼠标中键自动滚动带来 macOS 风格的自然滚动体验。

当你按下鼠标中键激活自动滚动后，向下移动鼠标会使页面向上滚动——与 macOS 用户习惯的"内容跟随手指"方向一致。

功能特性：
- 独立轴向控制——可分别反转垂直、水平方向，或同时反转
- 接近原生手感的加速曲线，带死区过滤
- 点击位置显示简洁的视觉指示器
- 中键点击即可开关（无需按住不放）
- 随时按 Esc、点击鼠标或切换窗口即可退出
- 零数据收集——设置仅存储在本地

适用人群：
- 在系统级别开启了自然滚动的 macOS 用户
- 经常在 Mac 和 Windows 之间切换的用户
- 觉得传统自动滚动方向不符合直觉的用户

适用于所有常规网页（http/https）。不会在 chrome:// 页面或 Chrome 网上应用店中运行。
