Reverse AutoScroll นำการเลื่อนแบบธรรมชาติสไตล์ macOS มาสู่ฟีเจอร์ออโตสกรอลด้วยคลิกปุ่มกลางของเมาส์ใน Chrome

เมื่อคุณคลิกปุ่มกลางเพื่อเปิดใช้งานออโตสกรอล การเลื่อนเมาส์ลงจะทำให้หน้าเว็บเลื่อนขึ้น — ตรงกับทิศทาง "เนื้อหาเคลื่อนตามนิ้ว" ที่ผู้ใช้ macOS คุ้นเคย

คุณสมบัติ:
- ควบคุมแต่ละแกนแยกอิสระ — กลับทิศแนวตั้ง แนวนอน หรือทั้งสองแกน
- เส้นโค้งความเร่งที่ให้ความรู้สึกเป็นธรรมชาติ พร้อมเดดโซน
- ตัวบ่งชี้ที่สะอาดตาแสดง ณ ตำแหน่งที่คลิก
- เปิด/ปิดด้วยคลิกปุ่มกลาง (ไม่ต้องกดค้าง)
- ออกได้ทุกเมื่อด้วย Esc, คลิกใดก็ได้ หรือเปลี่ยนหน้าต่าง
- ไม่เก็บข้อมูลใด ๆ — การตั้งค่าจัดเก็บในเครื่องเท่านั้น

เหมาะสำหรับ:
- ผู้ใช้ macOS ที่เปิดการเลื่อนแบบธรรมชาติทั้งระบบ
- ผู้ที่สลับใช้งานระหว่าง Mac และ Windows
- ผู้ที่รู้สึกว่าทิศทางออโตสกรอลแบบเดิมไม่เป็นธรรมชาติ

ใช้งานได้กับหน้าเว็บทั่วไปทุกหน้า (http/https) ไม่ทำงานบนหน้า chrome:// หรือ Chrome Web Store
