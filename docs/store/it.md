Reverse AutoScroll porta lo scorrimento naturale in stile macOS nello scorrimento automatico di Chrome attivato con il clic centrale.

Quando fai clic con il pulsante centrale per attivare lo scorrimento automatico, spostando il mouse verso il basso la pagina scorre verso l'ALTO — rispettando la direzione "il contenuto segue il dito" che gli utenti macOS si aspettano.

Funzionalità:
- Controllo indipendente per asse — inverti verticale, orizzontale o entrambi
- Curva di accelerazione naturale con zona morta
- Indicatore visivo discreto nella posizione del clic
- Attiva/disattiva con il clic centrale (non serve tenere premuto)
- Esci in qualsiasi momento con Esc, un clic qualsiasi o cambiando finestra
- Nessuna raccolta dati — impostazioni salvate solo in locale

Ideale per:
- Utenti macOS che hanno attivato lo scorrimento naturale a livello di sistema
- Chi passa spesso tra Mac e Windows
- Chi trova controintuitiva la direzione tradizionale dello scorrimento automatico

Funziona su tutte le normali pagine web (http/https). Non si attiva nelle pagine chrome:// né nel Chrome Web Store.
