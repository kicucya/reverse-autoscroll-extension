Reverse AutoScroll traz a rolagem natural do macOS para a rolagem automática com clique do meio do Chrome.

Ao clicar com o botão do meio para ativar a rolagem automática, mover o mouse para baixo rola a página PARA CIMA — seguindo a direção de "o conteúdo acompanha o dedo" que os usuários de macOS esperam.

Recursos:
- Controle independente por eixo — inverta o vertical, o horizontal ou ambos
- Curva de aceleração com zona morta de sensação nativa
- Indicador visual discreto na posição do clique
- Ative e desative com clique do meio (sem precisar segurar)
- Saia a qualquer momento com Esc, qualquer clique ou ao trocar de janela
- Zero coleta de dados — configurações armazenadas apenas localmente

Ideal para:
- Usuários de macOS que ativaram a rolagem natural em todo o sistema
- Quem alterna entre Mac e Windows
- Usuários que acham a direção da rolagem automática tradicional pouco intuitiva

Funciona em todas as páginas web comuns (http/https). Não é executado em páginas chrome:// nem na Chrome Web Store.
