Reverse AutoScroll은 macOS 스타일의 내추럴 스크롤을 Chrome의 휠 클릭 오토스크롤에 적용하는 확장 프로그램입니다.

휠 클릭으로 오토스크롤을 활성화하면, 마우스를 아래로 움직일 때 페이지가 위로 스크롤됩니다. macOS 사용자에게 익숙한 "콘텐츠가 손가락을 따라가는" 방향과 동일한 자연스러운 조작감을 제공합니다.

기능:
- 축별 독립 제어 — 수직, 수평, 또는 양쪽 모두 개별 반전 가능
- 데드존이 적용된 자연스러운 가속 커브
- 클릭 위치에 표시되는 깔끔한 시각적 표시기
- 휠 클릭으로 켜기/끄기 전환 (길게 누를 필요 없음)
- Esc 키, 아무 곳이나 클릭, 또는 창 포커스 이동으로 언제든 종료
- 데이터 수집 없음 — 설정은 로컬에만 저장

이런 분께 추천합니다:
- macOS에서 내추럴 스크롤을 시스템 전체에 적용하여 사용 중인 분
- Mac과 Windows를 번갈아 사용하시는 분
- 기존 오토스크롤 방향이 직관적이지 않다고 느끼시는 분

모든 일반 웹 페이지(http/https)에서 작동합니다. chrome:// 페이지 및 Chrome 웹 스토어에서는 작동하지 않습니다.
