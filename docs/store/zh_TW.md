Reverse AutoScroll 為 Chrome 的滑鼠中鍵自動捲動帶來 macOS 風格的自然捲動體驗。

當你按下滑鼠中鍵啟動自動捲動後，將滑鼠向下移動會讓頁面向上捲動——與 macOS 使用者習慣的「內容跟隨手指」方向一致。

功能特色：
- 獨立軸向控制——可分別反轉垂直、水平方向，或同時反轉
- 貼近原生手感的加速曲線，搭配死區過濾
- 點擊位置顯示簡潔的視覺指示器
- 滑鼠中鍵點擊即可切換（無須按住不放）
- 隨時按 Esc、點擊滑鼠或切換視窗即可退出
- 零資料收集——設定僅儲存於本機

適合對象：
- 在系統層級啟用了自然捲動的 macOS 使用者
- 經常在 Mac 和 Windows 之間切換的使用者
- 覺得傳統自動捲動方向不夠直覺的使用者

適用於所有一般網頁（http/https）。不會在 chrome:// 頁面或 Chrome 線上應用程式商店中運作。
