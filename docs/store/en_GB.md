Reverse AutoScroll brings macOS-style natural scrolling to Chrome's middle-click autoscroll.

When you middle-click to activate autoscroll, moving the mouse downwards scrolls the page UP — matching the "content follows finger" direction that macOS users expect.

Features:
- Independent axis control — reverse vertical, horizontal, or both
- Native-feel acceleration curve with dead zone
- Clean visual indicator at click position
- Toggle on/off with middle-click (no hold required)
- Exit at any time with Esc, any click, or window blur
- Zero data collection — settings stored locally only

Perfect for:
- macOS users who have enabled natural scrolling system-wide
- Anyone switching between Mac and Windows
- Users who find the traditional autoscroll direction counter-intuitive

Works on all regular web pages (http/https). Does not run on chrome:// pages or the Chrome Web Store.
