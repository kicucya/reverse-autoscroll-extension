Reverse AutoScroll привносит естественную прокрутку в стиле macOS в автопрокрутку Chrome по клику средней кнопкой мыши.

При активации автопрокрутки средней кнопкой мыши движение мыши вниз прокручивает страницу ВВЕРХ — в направлении «контент следует за пальцем», привычном для пользователей macOS.

Возможности:
- Независимое управление осями — инвертируйте вертикальную, горизонтальную ось или обе сразу
- Естественная кривая ускорения с мёртвой зоной
- Аккуратный визуальный индикатор в точке клика
- Включение/выключение по клику средней кнопкой (удерживать не нужно)
- Выход в любой момент по Esc, любому клику или при потере фокуса окна
- Никакого сбора данных — настройки хранятся только локально

Идеально подходит для:
- Пользователей macOS, включивших естественную прокрутку на уровне системы
- Тех, кто переключается между Mac и Windows
- Тех, кому традиционное направление автопрокрутки кажется неинтуитивным

Работает на всех обычных веб-страницах (http/https). Не работает на страницах chrome:// и в Интернет-магазине Chrome.
