Reverse AutoScroll は、macOS スタイルのナチュラルスクロールを Chrome のホイールクリックによるオートスクロールに適用する拡張機能です。

ホイールクリックでオートスクロールを起動すると、マウスを下に動かしたときにページが上方向にスクロールします。macOS ユーザーが慣れ親しんだ「コンテンツが指に追従する」方向と一致する、自然な操作感を実現します。

機能:
- 軸ごとの独立制御 — 垂直・水平・両方を個別に反転可能
- デッドゾーン付きの自然な加速カーブ
- クリック位置に表示されるすっきりとしたインジケーター
- ホイールクリックでオン/オフを切り替え（長押し不要）
- Esc キー、任意のクリック、ウィンドウのフォーカス移動でいつでも終了
- データ収集なし — 設定はローカルにのみ保存

こんな方におすすめ:
- macOS でナチュラルスクロールをシステム全体で有効にしている方
- Mac と Windows を行き来する方
- 従来のオートスクロール方向に違和感を覚える方

すべての通常の Web ページ（http/https）で動作します。chrome:// ページおよび Chrome ウェブストアでは動作しません。
