Reverse AutoScroll apporte le défilement naturel style macOS au défilement automatique par clic molette de Chrome.

Lorsque vous activez le défilement automatique par clic molette, déplacer la souris vers le bas fait défiler la page vers le HAUT — reproduisant le comportement « le contenu suit le doigt » auquel les utilisateurs macOS sont habitués.

Fonctionnalités :
- Contrôle indépendant des axes — inversez le défilement vertical, horizontal, ou les deux
- Courbe d'accélération naturelle avec zone morte
- Indicateur visuel discret à l'emplacement du clic
- Activation/désactivation par clic molette (pas besoin de maintenir enfoncé)
- Sortie à tout moment avec Esc, un clic quelconque, ou en quittant la fenêtre
- Aucune collecte de données — paramètres stockés localement uniquement

Idéal pour :
- Les utilisateurs macOS ayant activé le défilement naturel sur l'ensemble du système
- Toute personne alternant entre Mac et Windows
- Les utilisateurs trouvant le sens de défilement automatique classique peu intuitif

Fonctionne sur toutes les pages web standard (http/https). Ne s'exécute pas sur les pages chrome:// ni sur le Chrome Web Store.
